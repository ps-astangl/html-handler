# Installation 

Install Azure Function Apps Core Tools

```bash
npm install -g azure-functions-core-tools@3 --unsafe-perm true
```

Veryfy installation

```bash
func --version

```

Ensure Python Installed

```bash
python --verion
```

Install Azurite
```
npm install -g azurite
```

Start Azurite
```
azureite --silent
```
```

# Run

```bash
func start
```
